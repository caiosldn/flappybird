# BATMAN VOADOR GAME 

Este projeto implementa um jogo estilo Flap/Arcade com arquitetura Orientada a Objetos, sistema de ranking persistente em SQLite e log de erros, conforme os requisitos do Trabalho T2.

## 1. Instalação de Dependências

Para rodar o jogo, você precisa instalar a biblioteca PyGame. Abra seu terminal (Prompt de Comando ou PowerShell) na pasta raiz do projeto (`T2_FlappyBird/`) e execute o comando:

```bash
pip install pygame

python main.py
